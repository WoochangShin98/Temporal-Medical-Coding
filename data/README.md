# Dataset

This project uses the MIMIC-III dataset for clinical text and structured data.

Due to data usage restrictions and privacy policies, the full dataset is not included in this repository.

## Sample Data

We provide small sample files for demonstration purposes:

- sample/sample_radiology.csv
- sample/sample_discharge.csv

These samples contain a limited number of rows and truncated text fields.

## Data Access

To access the full dataset:

1. Complete PhysioNet credentialing
2. Request access to MIMIC-III
3. Download from: https://physionet.org/content/mimiciii/1.4/

## Tables Used

The following tables are used in this project:

- radiology
- discharge

## Notes

- Data is de-identified.
- Sample data is provided only for code demonstration and testing.
