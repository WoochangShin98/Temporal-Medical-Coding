Temporal Clinical Reasoning for Medical Coding

Spring 2026 CSCI 5541 NLP: Class Project - University of Minnesota

Stack OverFlowers

Abstract

This project predicts ICD and CPT codes from clinical records using temporal modeling. Unlike traditional methods that use a single document, our approach models multiple clinical notes over time using BioClinicalBERT and LSTM.

Approach

We compare two baselines and one temporal model:

  • Baseline 1: Single discharge summary (BioClinicalBERT)
  • Baseline 2: Multi-document (Longformer)
  • Temporal Model: BioClinicalBERT + LSTM with time ordering

Results

Model Micro F1 Macro F1 P@5 P@8
Baseline 1 0.4615 0.4660 0.2340 0.1603
Baseline 2 - - 0.1034 0.0767
Temporal Model 0.5335 0.4857 0.2357 0.1624

Conclusion

Temporal modeling significantly improves performance over single-document models, showing that modeling diagnostic progression across time is important.
