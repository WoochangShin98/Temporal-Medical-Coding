Temporal Clinical Reasoning for Medical Coding

Spring 2026 CSCI 5541 NLP: Class Project - University of Minnesota

Stack OverFlowers

Abstract

This project focuses on predicting ICD (diagnosis codes) and CPT (procedure codes) from clinical records. We model temporal clinical reasoning using BioClinicalBERT and LSTM.

Approach

We compare two baselines and one temporal model.

  • Baseline 1: BioClinicalBERT (single note)
  • Baseline 2: Longformer (multi-document)
  • Temporal Model: BioClinicalBERT + LSTM

Results

Model Micro F1 Macro F1 P@5 P@8
Baseline 1 0.4615 0.4660 0.2340 0.1603
Baseline 2 - - 0.1034 0.0767
Temporal Model 0.5335 0.4857 0.2357 0.1624

Conclusion

Temporal modeling improves performance and captures clinical progression.
