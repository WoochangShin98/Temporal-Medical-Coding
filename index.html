Abstract

This project proposes a temporal clinical reasoning framework for automatic ICD and CPT code prediction using electronic health records (EHRs). Unlike traditional approaches that process each clinical note independently, our method integrates multiple notes over time to capture the progression of a patient’s condition. Each note is encoded using BioClinicalBERT, followed by an LSTM-based temporal model to capture dependencies across notes. The framework jointly predicts ICD and CPT codes in a multi-label setting. Experiments on the MIMIC-III dataset demonstrate that temporal modeling improves predictive performance and consistency compared to single-note baselines.

Introduction / Motivation

Automated medical coding is essential in healthcare, supporting billing, clinical decision-making, and large-scale medical research.

However, existing systems process clinical notes independently and fail to capture temporal progression across patient records.

This motivates the need for temporal modeling across multiple notes.

Approach

We propose a temporal clinical reasoning framework that models multiple clinical notes over time using BioClinicalBERT and LSTM.

Dataset and Preprocessing

MIMIC-III dataset is used. Clinical notes are merged and sorted by time, and ICD codes are mapped.

Baseline Models

Baseline 1 uses single discharge notes. Baseline 2 uses merged documents with Longformer.

Temporal Model

Temporal model processes sequential notes using LSTM to capture progression across time.

Results

Temporal modeling improves ICD prediction (0.4615 → 0.5335). CPT prediction is more stable due to structured procedural patterns.
